## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/tfc-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd tfc-website
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```
